# Canine Club House Website

Static marketing site for **Canine Club House** – a family-owned dog daycare, boarding, and training business in Perry, Georgia.

Live site: https://www.canineclubhouseperry.com

---

## Tech Stack

- **HTML + CSS only**
- No JavaScript build tools, no frameworks
- Designed to be hosted on **GitHub Pages** (free)
